## 🎁 Konsep Website
Website ini dibuat seperti sebuah perjalanan kecil:
1. **Loading Chapter 22**
   * Pengguna melakukan tap untuk mengisi progress hingga 100%.

2. **Chapter 21**
   * Menampilkan foto wisuda dalam kartu interaktif.
   * Kartu dapat dibalik untuk membaca pesan ucapan.

3. **Open Memory Box**
   * Kotak kenangan yang ketika dibuka akan menampilkan kumpulan foto polaroid.

4. **Pesan & Ucapan**
   * Berisi ucapan kelulusan dan ulang tahun yang personal.

5. **Celebration**
   * Efek konfetti dan animasi penutup sebagai perayaan.

## 🌐 Deploy
### GitHub Pages
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin <repository-url>
git push -u origin main

Kemudian aktifkan GitHub Pages melalui:
Settings → Pages → Deploy from Branch

### Netlify
1. Login ke Netlify
2. Pilih "Add New Site"
3. Drag & Drop folder project
4. Website langsung online

### Vercel
vercel --prod
